Delivering a Custom Font to the Client

This page demonstrates how to deliver a custom font to a client's browser. The font used here is called "Nosifer", which is likely not installed on most computers by default.

How the Custom Font Was Included

To include the "Nosifer" font in this HTML file, the following steps were taken:

  1. A link to the Google Fonts stylesheet was added in the <head> section: 
    <link href='https://fonts.googleapis.com/css?family=Nosifer' rel='stylesheet'>
  2. The CSS font-family property was set to "Nosifer" in the <style> block within the <head> section: 
    body {
    font-family: 'Nosifer';
    font-size: 22px;
}
  3. The font was then applied to the entire body of the HTML document, ensuring that all text elements use the "Nosifer" font.
